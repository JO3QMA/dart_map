## ダーツの旅（dart_map）

ランダムに選ばれた地点をもとに、バーチャルな「ダーツの旅」を楽しむ Web アプリケーションです。

### 利用データについて

本アプリケーションでは、住所情報として **Geolonia 住所データ** を利用しています。
